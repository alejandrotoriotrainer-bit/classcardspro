🔥 Configurar Firebase

Rellena los datos de tu proyecto Firebase. Los encuentras en ⚙️ Configuración del proyecto → General → Tu app web.

  1. Ve a console.firebase.google.com → entra con Google
  2. Clic en Crear proyecto → nombre → sin Analytics → Crear
  3. Menú izquierdo → Compilación → Realtime Database → Crear base de datos → Modo de prueba → Activar
  4. Menú izquierdo → ⚙️ Configuración del proyecto → General → baja a Tus apps → icono </> → registra con cualquier nombre
  5. Verás los valores apiKey, databaseURL, etc. Cópialos uno a uno abajo

Publica en GitHub Pages

Para que la cámara funcione en móvil la app necesita estar en HTTPS.

📋 Guía: publicar en GitHub Pages

  1. Descarga el archivo .html con el botón de arriba (ya tiene tu config de Firebase guardada)
  2. Ve a github.com → crea cuenta gratis si no tienes
  3. Clic en New repository → nómbralo classcards → público → crear
  4. Sube el archivo: clic en Add file → Upload files → arrastra el .html → renombralo a index.html → Commit
  5. Ve a Settings → Pages → Source: Deploy from a branch → Branch: main → carpeta: / (root) → Save
  6. En 2 minutos tu app estará en: https://TU_USUARIO.github.io/classcards
  7. Abre esa URL en tu tablet y móvil. La cámara ya funcionará y los datos se sincronizan en tiempo real vía Firebase 🎉
